### Инструкция по использованию Jdoodle

Jdoodle - онлайн-компилятор кода. Чтобы начать работу:
1. перейдите на сайт [https://www.jdoodle.com/](https://www.jdoodle.com/);
2. авторизуйтесь по кнопке _Login_:
3. вернитесь на главную страницу и выберите _Let's code_;
4. выберите язык программирования и нажмите _Start coding_.

Пишите код в открывшемся редакторе. Полезные кнопки:

- _Execute_ запускает код;
- _Debug with JDroid_ запускает отладчик;
- _Save_ сохраняет проект;
- _Project share_ открывает окно доступов.
 
Чтобы получить ссылку на проект в Jdoodle, нажмите  _Project share_, выберите _Anyone with the link_ в General access. Нажмите _Copy link_, чтобы скопировать ссылку.
